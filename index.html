📋 このアプリのデータ構造

すべてのデータはこのHTMLファイル内の DB オブジェクトに直接記載されています。テキストエディタ（メモ帳、VSCode等）でファイルを開き、編集するだけでメンテナンスができます。

✏️ 新しいライブを追加する方法

ファイルをテキストエディタで開き、// === DB END === の直前に以下の形式で追加してください：

{ date: "2026-08-01", // 日付 YYYY-MM-DD形式 title: "岡崎BOPPERSライブ", // イベント名 venue: "BOPPERS", // 会場名（統一して入力） setlist: [ { artist: "BOOWY", title: "B BLUE" }, { artist: "ELLEGARDEN", title: "風の日" }, // ... 曲を追加 ], bands: [ "バンドA", "バンドB", // ... 共演バンドを追加（SLM48は不要） ] },

🔄 既存ライブを修正する方法

ファイル内で日付（例: 2025-01-26）を検索して該当箇所を見つけ、setlistやbandsを直接編集してください。

🗑️ ライブを削除する方法

該当ライブの { date: "..." } から始まる波カッコ一組をまるごと削除し、前後のカンマも整理してください。

💾 バックアップのすすめ

編集前にファイルをコピーしてバックアップを取ることをおすすめします。また、このファイルはブラウザで直接開けるので、USBメモリや Google Drive で管理すると便利です。

📱 スマホで使う場合

Google Drive などにアップロードしてブラウザで開くと、スマホでも検索・閲覧が可能です。
